# Notes App — Backend API

Built this while learning Django REST Framework.
Goal is to become a full Stack Application Developer

## What I Built
A notes API where users can register, login and manage 
their notes with subtasks. Each user can only see their 
own notes.

## Stack
- Django REST Framework
- SimpleJWT
- SQLite

## Where I Struggled
- Nested serializers took me a long time to understand
- Pagination had a lot of bugs before it worked
- JWT logout and blacklisting was confusing at first

## What I Learned
- How REST APIs work end to end
- JWT authentication flow
- How to handle nested data (notes with subtasks)
